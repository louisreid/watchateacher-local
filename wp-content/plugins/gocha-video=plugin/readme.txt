== Description ==

In many places there are very thoughtful and practical ways to relate to video content. Now, WordPress is on that list, too – all thanks to our plugin. It is a must-have if you publish a lot of videos with traffic around them.

Do you create videos on a daily basis? Do you share them online? Are you a WordPress addict? We have something for you.

http://www.gochavideo.com/

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/gocha-video-plugin` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the Appearance -> Customize -> GOCHA Video screen to configure the plugin
4. Check documentation for more info: [GOCHAVideoPluginManual](http://gochavideo.com/manual/ “Manual for GOCHA Video Plugin”)
